VOICE RECORDING REPOSITORY PLUGIN
=================================

Voice recording is a Moodle 2.X repository that allow users to record and upload audio files inside Moodle using NanoGong applet 

Web: https://github.com/IOC/moodle-repository-voicerecording


Install
=======
Download code from gitHub, and copy inside directory <moodle directory>/repository/voicerecording


Author
======

Marc Català Sala <mcatala@ioc.cat>


NanoGong
========

This repository plugin requires NanoGong Applet, you can download source code from http://gong.ust.hk/nanogong/downloads/nanogong-4.2-src.zip
